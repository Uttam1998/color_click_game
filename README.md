# 🎨 Color Click Game

A fun and interactive browser game built using **HTML, CSS, and Vanilla JavaScript**.

Click the correct color before time runs out and try to score as high as possible!

This project is designed to practice:

- DOM Manipulation
- Event Listeners
- setInterval & Timers
- State Management
- Working with dataset
- Basic Game Logic
